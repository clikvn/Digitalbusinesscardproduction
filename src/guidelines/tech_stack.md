# Tech Stack

## Frontend

**Framework & Language:**
- React 18.x
- TypeScript 5.x
- Vite (build tool)

**UI & Styling:**
- Tailwind CSS 4.0
- Shadcn UI (headless components)
- Lucide React (icons)

**State Management:**
- TanStack Query 5.x (server state)
- React Hooks (local state)

**Data Visualization:**
- Recharts 2.x

**Forms:**
- React Hook Form 7.55.0

**Routing:**
- React Router

---

## Backend

**Backend-as-a-Service:**
- Supabase
  - PostgreSQL 15+
  - Supabase Auth
  - Supabase Storage
  - Row Level Security (RLS)

**Architecture:**
- Direct client calls (NO Edge Functions)
- RLS for security
- Database functions for initialization
